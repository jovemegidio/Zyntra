Como Cadastrar um Fornecedor

Cadastre fornecedores com dados fiscais, endereço, contatos e categorias de materiais.

Introdução

O cadastro de fornecedores centraliza todos os dados necessários para cotações, pedidos e entrada de notas fiscais. Mantenha os dados atualizados para evitar problemas fiscais.

Passo 1 — Acessar Fornecedores

No módulo Compras, clique em Fornecedores na barra lateral.
