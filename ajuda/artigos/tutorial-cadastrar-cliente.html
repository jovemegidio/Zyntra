Como Cadastrar um Novo Cliente

Guia completo para cadastrar novos clientes no módulo de Vendas do Aluforce, incluindo dados fiscais, endereço e contatos.

O cadastro de clientes é essencial para a operação de vendas na Aluforce. Cada cliente deve ter seus dados fiscais, endereço de entrega e contatos devidamente cadastrados antes de criar pedidos ou orçamentos.

Pré-requisitos

  • Ter acesso ao módulo de Vendas
  • Ter em mãos os dados fiscais do cliente (CNPJ, Razão Social, I.E.)

Passo 1 — Acesse a lista de clientes

No menu lateral, clique em Clientes (ícone de usuários). A lista com todos os clientes cadastrados será exibida. Aqui você pode pesquisar clientes existentes antes de criar um novo.

Passo 3 — Preencha os dados fiscais

Na aba Dados Fiscais, preencha as informações obrigatórias: CNPJ, Razão Social e Inscrição Estadual. Ao digitar o CNPJ, o sistema pode preencher automaticamente os dados da empresa via consulta à base da Receita Federal.

Consulta automática por CNPJ

Ao digitar o CNPJ completo e sair do campo, o sistema consulta automaticamente a Receita Federal e preenche Razão Social, Nome Fantasia e endereço. Revise os dados antes de salvar.

Passo 4 — Preencha o endereço

Clique na aba Endereço e preencha o CEP. O sistema preencherá automaticamente Logradouro, Bairro, Cidade e UF. Complete com o Número e Complemento, se necessário.

Passo 5 — Adicione os contatos

Na aba Contatos, adicione pelo menos um contato principal com Nome, E-mail e Telefone. Você pode adicionar múltiplos contatos (ex: comprador, financeiro, recebimento).

Passo 6 — Salve o cadastro

Após preencher todas as abas, clique no botão Salvar para concluir o cadastro do cliente. O sistema validará os dados e exibirá uma confirmação de sucesso.

CNPJ duplicado

Se o CNPJ informado já existir no sistema, o cadastro não será permitido. Verifique se o cliente já está cadastrado antes de criar um novo registro.
