Como Emitir Carta de Correção (CC-e)

Aprenda a emitir uma Carta de Correção Eletrônica para corrigir informações de NF-e já autorizadas.

Introdução

A Carta de Correção Eletrônica (CC-e) é um documento fiscal utilizado para corrigir informações complementares de uma NF-e já autorizada na SEFAZ, sem a necessidade de cancelamento. Na Aluforce, erros em dados como informações adicionais, dados da transportadora ou dados do vendedor podem ser corrigidos via CC-e.

É importante ressaltar que a CC-e possui limitações — não é possível corrigir valores, quantidades, dados do destinatário ou CFOP.

Pré-requisitos

  • Acesso ao módulo NF-e — Permissão para emitir Carta de Correção.
  • NF-e autorizada com erro em dados complementares — A NF-e deve estar com status "Autorizada".

Informação: Limite de CC-e por NF-e

É possível emitir até 20 Cartas de Correção para a mesma NF-e. A CC-e mais recente sempre substitui as anteriores.

Atenção: Limitações da CC-e

A CC-e NÃO pode corrigir: valores, quantidades, dados do destinatário (CNPJ, razão social), alíquotas de impostos ou CFOP. Para esses campos, é obrigatório o cancelamento e emissão de nova NF-e.

O que pode e não pode corrigir via CC-e

SituaçãoCampoPode corrigir?
Dados adicionaisInformações complementares, observações Sim

CFOPCódigo Fiscal de Operações Não
