🔧 Correção do Erro SSL_PROTOCOL_ERROR

⚠️ Problema Detectado

Seu navegador está tentando usar HTTPS mas o servidor está em HTTP (desenvolvimento).

Causa: Cache HSTS (HTTP Strict Transport Security) forçando HTTPS.

1 Limpar HSTS no Chrome/Edge
  • Abra uma nova aba e digite:
chrome://net-internals/#hsts
  • Na seção "Delete domain security policies":
  • Digite: localhost → Clique em Delete
  • Digite: 192.168.68.127 → Clique em Delete
2 Limpar Cache Completo
  • Pressione: Ctrl + Shift + Del
  • Selecione: "Todo o período"
  • Marque:
  • ✅ Cookies e outros dados de sites
  • ✅ Imagens e arquivos em cache
  • Clique em "Limpar dados"
3 Fechar e Reabrir o Navegador

Feche TODAS as janelas do navegador e abra novamente.

4 Acessar via HTTP

Use HTTP (não HTTPS):

http://localhost:3000

OU

http://192.168.68.127:3000
✅ Acessar ALUFORCE Agora

💡 Alternativa Rápida

Se o problema persistir, use o navegador em modo anônimo:

  • Chrome/Edge: Ctrl + Shift + N
  • Firefox: Ctrl + Shift + P

Isso ignora completamente o cache e HSTS.

✅ Correções Aplicadas no Servidor:
  • HSTS desabilitado em desenvolvimento
  • CSP desabilitado em desenvolvimento
  • Cross-Origin headers configurados
  • Servidor rodando em HTTP puro
