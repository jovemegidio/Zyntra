# Templates de Importação - Módulo Financeiro

Esta pasta contém os templates Excel (XLSX) para importação de dados no módulo financeiro.

## Templates Disponíveis:

1. **template_contas_pagar.xlsx** - Para importação de contas a pagar
2. **template_contas_receber.xlsx** - Para importação de contas a receber
3. **template_bancos.xlsx** - Para importação de contas bancárias
4. **template_movimentacoes.xlsx** - Para importação de movimentações bancárias
5. **SGE_Fluxo_Caixa.xlsx** - Para importação de fluxo de caixa

## Instruções:

1. Baixe o template correspondente
2. Preencha os dados seguindo o formato das colunas
3. Não altere os cabeçalhos das colunas
4. Salve o arquivo e faça o upload no sistema

## Formato de Datas:
Use o formato: DD/MM/AAAA (ex: 31/01/2026)

## Formato de Valores:
Use números com duas casas decimais (ex: 1500.00)
