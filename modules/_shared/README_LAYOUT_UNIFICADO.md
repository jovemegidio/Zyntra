# ALUFORCE v2.0 - Sistema de Layout Unificado## Visão GeralO sistema de layout unificado garante consistência visual em todos os módulos do ALUFORCE, seguindo o padrão profissional inspirado no Omie, mas com uma versão mais fina e elegante.## Arquivos do Sistema```modules/_shared/├── aluforce-layout.css      # CSS principal do layout├── aluforce-layout.html     # Template HTML do header/sidebar├── aluforce-layout.js       # JavaScript para inicialização└── themes/    ├── theme-compras.css    # Tema azul claro (Compras)    ├── theme-vendas.css     # Tema azul (Vendas)    ├── theme-pcp.css        # Tema vermelho (PCP)    ├── theme-financeiro.css # Tema verde (Financeiro)    ├── theme-rh.css         # Tema roxo (RH)    └── theme-nfe.css        # Tema laranja (NF-e/Logística)```## Como Usar### 1. Incluir os arquivos CSS no `<head>````html<head>    <!-- Font Awesome (ícones) -->    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.2/css/all.min.css">        <!-- Layout Base -->    <link rel="stylesheet" href="/modules/_shared/aluforce-layout.css">        <!-- Tema do Módulo (escolha um) -->    <link rel="stylesheet" href="/modules/_shared/themes/theme-compras.css"></head>```### 2. Estrutura HTML do Body```html<body class="alu-layout">    <div class="alu-app-container">        <!-- Sidebar -->        <aside class="alu-sidebar">            <!-- Conteúdo da sidebar (ver template) -->        </aside>                <!-- Header -->        <header class="alu-header">            <!-- Conteúdo do header (ver template) -->        </header>                <!-- Área Principal -->        <main class="alu-main-area">            <!-- Barra de Filtros (opcional) -->            <div class="alu-filter-bar">                <!-- Filtros -->            </div>                        <!-- Conteúdo -->            <div class="alu-content-area">                <!-- Seu conteúdo aqui -->            </div>        </main>    </div>        <!-- JavaScript -->    <script src="/modules/_shared/aluforce-layout.js"></script></body>```### 3. Incluir o JavaScript antes de `</body>````html<script src="/modules/_shared/aluforce-layout.js"></script>```## Componentes Disponíveis### Sidebar (52px de largura)```html<aside      { id: 'dashboard', name: 'Dashboard' },        { id: 'pedidos', name: 'Pedidos', closable: true }    ]});```### Métodos Disponíveis```javascript// Adicionar tabAluforceLayout.addTab({ id: 'nova', name: 'Nova Tab', closable: true });// Remover tabAluforceLayout.removeTab('nova');// Atualizar badgeAluforceLayout.updateBadge('badge-orders', 5);// Mostrar notificaçãoAluforceLayout.setNotificationDot(true);// LogoutAluforceLayout.logout();```### Eventos Customizados```javascript// Navegação na sidebardocument.addEventListener('alu-navigate', (e) => {    console.log('Seção:', e.detail.section);});// Mudança de tabdocument.addEventListener('alu-tab-change', (e) => {    console.log('Tab:', e.detail.tabId);});// Buscadocument.addEventListener('alu-search', (e) => {    console.log('Query:', e.detail.query);});```## Cores por Módulo| Módulo | Cor Principal | Tema ||--------|--------------|------|| (>1024px)**: Layout completo- **Tablet (768-1024px)**: Busca escondida, info do usuário simplificada- **Mobile (<768px)**: Sidebar escondida, menu hamburger## Migração de Módulos Existentes1. Fazer backup do arquivo original2. Adicionar os CSS no `<head>`3. Adicionar classe `alu-layout` no `<body>`4. Substituir sidebar e header pelo template5. Envolver conteúdo em `alu-main-area` e `alu-content-area`6. Incluir o JavaScript antes de `</body>`7. Testar e ajustar estilos específicos do módulo
